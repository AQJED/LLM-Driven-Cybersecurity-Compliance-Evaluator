LLM‑Driven Cybersecurity Compliance Evaluator

Python‑Based Tool for Automated NCA Compliance Assessment

Assess your organization’s cybersecurity controls against National Cybersecurity Authority requirements using a conversational AI backend.

Results are processed from CSV input and exported to Excel for easy reporting.

Overview

This tool uses an LLM to compare company cybersecurity controls against NCA compliance requirements and returns standardized assessment results.

Features

  • Automated compliance assessment via AI interaction
  • Reads requirements & controls from CSV files
  • Exports assessment results to Excel
  • Interactive UI for file loading & report generation
  • Real‑time analysis using the Ollama API

How to Run

  • Install Python dependencies: pip install pandas ollama
  • Launch the UI and load your control & requirement CSV files
  • Click ‘Evaluate’ to trigger AI assessment
  • Download the Excel report
